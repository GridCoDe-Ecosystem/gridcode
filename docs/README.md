# GridCoDe Documentation

This directory contains all formal documentation for the GridCoDe protocol.

## Folder Guide

- **architecture/** — Runtime and execution architecture
- **build/** — Build plans and implementation strategy
- **contracts/** — Contract-level specifications
- **economics/** — Economic model and incentive logic
- **governance/** — Governance framework and DAO logic
- **security/** — Threat models and guarantees
- **specifications/** — Canonical protocol specifications

All documents in this directory must preserve version parity with their signed source.
