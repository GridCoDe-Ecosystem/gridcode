# Whitepaper Documents

This directory contains public-facing, high-level documents describing the GridCoDe system.

## Status

Documents in this directory are **not part of the MDCR-controlled specification set**.

They serve as:

- conceptual architecture references  
- public whitepapers  
- system-level explanations  

## Implementation Guidance

All implementation MUST follow the MDCR-controlled documents located in:

- /docs/core/
- /docs/runtime/
- /docs/store/

Whitepapers do not override protocol specifications.
